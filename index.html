What is Polygon Staking?

Polygon Staking is the official dashboard for delegating POL to Polygon Chain validators; the Polygon docs cover delegation, rewards, restaking, unbonding, and stake management.

How Polygon Staking works

  1. Connect walletUse an Ethereum mainnet wallet with POL and enough ETH for gas.
  2. Choose a validatorOpen the dashboard, review the validator list, and select Delegate on the validator you want.
  3. Approve and delegateEnter the POL amount, approve the token transaction, then confirm the delegation transaction in your wallet.
  4. Manage rewardsUse My Account to withdraw POL rewards, restake rewards, move stake, or start unbonding.

Before you stake

Official route

Use staking.polygon.technology when connecting a wallet and verify the domain before signing.

Validator selection

Delegators choose validators directly; rewards can vary with stake weight, validator commission, checkpoints, and network conditions.

Ethereum custody model

Staked funds are locked in contracts on Ethereum mainnet, and validators do not take custody of delegator tokens.

Polygon Staking FAQ

What is polygon staking?

It is the official Polygon dashboard flow for delegating POL to validators, viewing delegations, withdrawing rewards, restaking rewards, moving stake, and unbonding.

Does polygon staking happen on Ethereum mainnet?

Yes. Polygon documentation states that staking transactions take place on Ethereum mainnet for security, so users need ETH for gas.

Which token is used for polygon staking?

POL is the native gas and staking token for Polygon Chain, replacing MATIC; Polygon docs also note that MATIC delegations receive rewards in POL.

Is there a minimum for polygon staking?

Polygon docs state there is no minimum delegation amount, though users can start with 1 POL token.

How long does polygon staking unbonding take?

The unbonding period is 80 checkpoints; each checkpoint is approximately 30 minutes, but timing can vary with Ethereum congestion and validator consensus.
